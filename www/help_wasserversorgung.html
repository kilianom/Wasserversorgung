1. Suche nach Ort
  2. Messen von Strecken und Flächen
  3. Weideflächen erfassen
    • Im Uhrzeigersein Ecken der Weideflächen anklicken
    • Klick auf ‘Finish’ oder erste Ecke schließt Eingabe der Parzelle ab
  4. Tränken erfassen
    • Klick auf den Standort der Tränke
  5. Eingabe verändern
    • Ecken der Parzellen oder Tränken können durch Drag and Drop verändert werden
    • Eingabe mit Klick auf ‘Save’ bestätigen
  6. Parzellen oder Tränken löschen
    • Klick auf das zu löschende Objekt
    • Eingabe mit Klick auf ‘Save’ bestätigen
  7. Hilfe anzeigen
  8. Wasserversorgung erfassen und in Ansichtsmodus wechseln

Im Ansichtsmodus kann die Karte als Bild heruntergeladen werden. Für eine spätere Nutzung können die Eingaben zu Parzellen und Tränken im Browser gespeichert werden. Gespeicherte Eingaben können manuell aus dem Browserspeicher entfernt werden. Das Speichern im Browser funktioniert in den meisten aktuellen Browsern. Durch Klick auf ‘Bearbeiten’ können die Eingaben weiter verändert werden. Informationen zur Wasserversorgung finden Sie hier.
